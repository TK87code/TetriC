# TetriC
![GIF](https://github.com/TK87code/TetriC/blob/master/Gameplay.gif)

## Overview
Tetris clone written in C with Raylib. This is created for educational purposes.
## Requirement
Windows 10
MSVC
Raylib
## Usage
Press Arrow keys to move tetrimino. Press Z to rotate.
## Features
Nothing special
